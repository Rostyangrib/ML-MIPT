# Лабораторные работы по курсу машинного обучения

### Лабораторная работа 1
**Цель:** Предсказать цену алмаза на рынке, по его физическим характеристикам.             
**Методы:** Линейная регрессия, L1, L2, Elastic-net регуляризации.

### Лабораторная работа 2
**Цель:** Предсказать вероятность победы сил света в игре DOTA2, по игровой статистике.                  
**Методы:** Ансамблевые модели, Стекинг, Блендинг. 

### Лабораторная работа 3
**Цель:** Выполнить детекцию основыных деталей мордочки кота.                      
**Методы:** Базовая сверточная сеть, ResNet, BatchNorm.
